c++梳理（八）

viat · 2021-4-27 · 次阅读

函数重载 发生在一个作用域里面

父子兼容

子类对象能当父类对象用
1.子类对象可以初始化父类对象（子类会退化为父类）
2.子类对象可以赋值给父类对象
3父类指针可以指向子类
4.父类引用可以直接引用子类

1.子类对象可以初始化父类对象（子类会退化为父类）
2.父类对象或者父类指针只能调用父类成员
3.可以直接访问被隐藏的方法

多继承成员冗余

virtual +返回值 + 函数名（）{}此为虚函数 用于实现多态
virtual 返回值 + 函数名（）= 0 此为纯虚函数 做接口用的

多态

多态：
1.在程序运行中表现出来的动态特性。
2.子类中函数重写要实现多态
3.多态是面向对象设计的基础
多态写法 ：父类中加virtual，子类可加可不加，但是。
virtual +返回值 + 函数名（）{}
多态的意义就是传父类对象调用父类相关函数，传子类对象调用子类相关函数
多态是在程序运行中表示出来的
对一个父子类都有的函数，在子类中重写，就要实现多态

静态联偏：编译的时候要用的函数已经确定了 表现在函数的重载
动态联偏：运行的时候才去确定具体去用哪个函数

重写和重写的区别

方法重写(overriding)：
　　1、也叫子类的方法覆盖父类的方法，要求返回值、方法名和参数都相同。
　　2.子类方法的的访问级别不能低于父类相应方法的访问级别
方法重载(overloading):
1.重载是在同一个类中的两个或两个以上的方法，拥有相同的方法名，但是参数却不相同，方法体也不相同，最常见的重载的例子就是类的构造函数
