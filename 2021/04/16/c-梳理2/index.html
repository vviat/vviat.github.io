c++梳理（二）

viat · 2021-4-16 · 次阅读

构造函数

对自己定义的方法进行初始化；（从无到有构造一个对象）
构造函数是定义在类内的特殊函数，它用于描述对类进行初始化的算法；
我们不需要手动调用构造函数，当我们创建对象的时候，构造函数会被自动调用；

默认构造函数

如果不声明任何构造函数，系统会生成默认构造函数；
默认构造函数的特点：

  1. 参数列表为空，不为数据成员赋初值；
  2. 如果类内定义了成员的初始值，则使用类内定义的初始值；
  3. 如果没有定义类内的初始化，则以默认方式初始化；
  4. 基本数据类型的数据默认初始化的值是不确定的；

构造函数的性质

  1. 函数名与类名完全相同
  2. 不能定义返回值，也不能有return语句；
  3. 形参可有可无，也可带有默认参数；
  4. 可以重载；

复制构造函数

生成一个对象的副本有两种途径：

  1. 建立一个新对象 ，将一个已有对象的数据成员值取出来，赋值给新对象。
  2. 利用复制构造函数，通过一个已有的对象，来执行一个新对象的构造。
    复制构造函数具有一般构造函数的所有特点；
    复制构造函数的形参是对本类的一个对象的引用，目的是用一个已经存在的对象来初始化一个新的对象。（引用传递）
    实际工作中，根据需求要定义一个复制构造函数，实现同类对象直接的数据成员的传递。
    如果没有定义复制构造函数，系统会自动生成一个复制构造函数。
    `
    class 类名{
    public:
    类名（类名＆　对象名）{
    //实现
    }
    };

class 类名{//深拷贝
public:
类名（const 类名＆　对象名）{
p = new p();
p = (r.p);
}
private:
int *p;
};


### 复制构造函数的调用
1. 当用类的一个对象去初始化该类的另一个对象的时候

Point a(1,2);//有参的构造函数
Point b(a);//用对象ａ去初始化对象ｂ，调用复制构造函数
Point c = b;//用对象ｂ去给对象ｃ赋值，调用赋值构造函数

2. 当函数的形参是一个类的对象，调用函数时进行形实结合的时候

void f(Point p){}
Point a(1,2);
f(a);

3. 当函数的返回值是类的对象的时候；

Point g(){
Point a(1,2);
return a;
//当函数返回值的时候，局部变量销毁;
//return a返回的并不是ａ这个对象的本身，而是通过复制构造函数用ａ重新构造的对象
//在函数返回后，临时变量ａ销毁。
}
`

为什么系统提供复制构造函数了，我们还有实现；

很多情况下，我们没有必要去定义一个复制构造函数，但如果我们不想把一个对象全部复制，我们想要有选择的复制，或者在复制的过程中加点什么（有变化的复制），此时我们就需要编写新的复制构造函数；
除此之外，如果类中成员有指针类型，默认的构造函数实现的只是浅拷贝，存在安全隐患，此时我们就需要深拷贝；
有指针会把原有的对象的地址和新对象的地址指向同一区域，这样在改一个值的时候，另一个也会改，析构的时候一个对象会被析构两次，造成程序崩溃。
